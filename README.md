# 📊 Sales & Customer Performance Analysis (SQL)

## 📌 Project Overview
This project analyzes retail sales data using MySQL to uncover key business insights such as revenue drivers, profitability trends, discount impact, and regional performance.

The goal is to simulate real-world Business Analyst decision-making using SQL techniques like aggregation, joins, and subqueries.
